# Help Desk Heroes
Doctors &amp; Nurses keep getting tech problems. You are the IT help desk. Read each ticket, then clicking the matching tool to fix it in time. Every fix shows how IT keeps patients safe. 3 wrong answers and your shift is done!

🎮 About

We hear a lot about doctors and nurses but behind every chart, vitals monitor, and
medication scan, there's an IT team keeping it all online. Help Desk Heroes is a quick,
arcade-style game that puts you in that seat.

Tickets pop up from clinical staff with real-world tech problems. Read each one, pick the
right IT tool before the timer runs out, and learn one concrete way IT keeps patients safe
with every fix. Miss too many and the shift is over.

It's a single HTML file, runs entirely in the browser, needs no internet and no
install, and works on desktop and mobile (mouse or touch).

🕹️ How to Play

1. Click anywhere to start your shift.
2. A ticket appears from a doctor or nurse (e.g. "I'm locked out of my account before rounds!").
3. Click the matching IT tool from the toolkit along the bottom before the timer empties.
4. ✔ Correct fix → score + a quick lesson on how IT supports care.
   ✘ Wrong tool or timeout → you lose a ❤️ Care point.
6. Three missed/wrong fixes and the shift ends. Clear tickets to level up, they arrive faster.

💡 Faster fixes earn bonus points. Speed and accuracy make the difference.
